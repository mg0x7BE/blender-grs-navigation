# Blender GRS Navigation

Single-file Blender addon. G/R/S switch the active tool (move, rotate, scale) instead of triggering modal transforms.

Tested on Blender 5.0.

## Install

1. Download `blender_grs_navigation.py`
2. Edit > Preferences > Add-ons > "Install from Disk..."
3. Don't forget to enable "Blender GRS Navigation"
4. Save Preferences
